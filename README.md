# PackFlow ERP - Frontend UI 📦

Interfaz de usuario moderna, ejecutiva y de alta fidelidad diseñada para la administración y operación de plantas de manufactura de empaques y cajas.

## Características Principales
- **Panel de Cotizaciones Técnicas:** Formulario dinámico e inteligente que calcula costos en tiempo real según las variables geométricas de la caja.
- **Diseño Corporativo:** Interfaz limpia construida bajo estándares de diseño ejecutivo utilizando componentes optimizados.
- **Gestión Modular:** Vistas dedicadas para la administración de catálogos de clientes, sucursales y cuentas bancarias.

## Stack Tecnológico
- **Librería Core:** React.js
- **Sistema de Diseño:** Material UI (MUI) mediante el uso extensivo de la propiedad `sx` para maquetación avanzada.
- **Estilos:** Emotion / Styled Components (MUI Built-in)
- **Cliente HTTP:** Axios para comunicación fluida con la API.
